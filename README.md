## Overview

Package for working with Input Output Tables in quantitative economics.

Download and convert I-O-data from different sources.

Currently supported databases:

-   WIOD:
    -   download tables
    -   convert to standardized long format

Planned support:

-   Eora
-   OECD I-O-tables
-   USITC ITPD-IO
-   Eurostat
